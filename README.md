# Delphi Display
This is the main display that everyone in a game watches.

## Goals
- Support for most browsers, especially smart TVs
- 16:9 oriented UI
- Instant loading

## Develop

1. Install dependencies
```
yarn install
```

2. Serve local version
```
yarn serve
```
